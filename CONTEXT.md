# free-ai

A minimal Persian-language app that displays a curated list of platforms offering free (no-cost) AI access. Each platform is documented in a markdown file with a link, description, and metadata.

## Language

**Platform**:
A service or website that provides free access to AI capabilities (chat, generation, etc.).
_Avoid_: Source, provider, service

**Free**:
No monetary cost to use. Rate limits, signup requirements, or feature restrictions may still apply.
_Avoid_: Open, public, complimentary

**Link**:
The URL pointing to the platform's website or access point.
_Avoid_: URL, address, source

**Entry**:
A markdown file that documents a single platform, stored in `content/`. Contains frontmatter fields (title, lastUpdated, link, author, tested, isSafe) and a body with the platform description.
_Avoid_: Card, item, page

**Author**:
A person who contributed the platform entry. Represented as a link to their Twitter/X profile.
_Avoid_: Contributor, submitter

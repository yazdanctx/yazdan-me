In this first part, we'll cover the core type system.

## Primitive Types

TypeScript extends JavaScript with type annotations:

```typescript
let name: string = "hello";
let age: number = 30;
let isActive: boolean = true;
```

## Interfaces

Interfaces define the shape of objects:

```typescript
interface User {
  name: string;
  age: number;
}
```

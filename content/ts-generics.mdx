Generics allow you to write reusable components that work with any type.

## Basic Generics

```typescript
function identity<T>(arg: T): T {
  return arg;
}
```

## Generic Constraints

You can constrain generics to specific shapes:

```typescript
function getLength<T extends { length: number }>(arg: T): number {
  return arg.length;
}
```
